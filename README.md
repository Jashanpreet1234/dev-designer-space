## 🚀 **Quick Launch Instructions**

### **🔧 System Requirements**
- 💻 **Node.js** 16.0.0+
- 📦 **npm** 8.0.0+
- 🌐 **Modern Browser** (Chrome 90+, Firefox 88+, Safari 14+)

### **⚡ Installation & Launch**

   ```bash
# 🛸 Clone the space station
git clone https://github.com/Jashanpreet1234/portfolio.io-main.git
cd portfolio.io-main

# 🔋 Power up the systems
   npm install

# 🚀 Launch the development server
   npm run dev

# 🌌 Access Mission Control at
# http://localhost:3000
```

### **🖥️ Windows Quick Launch**
```batch
# One-click deployment for Windows
deploy.bat
```

## 🚀 **Deployment Options**

### **☁️ Cloud Platforms**

<details>
<summary><strong>🔥 Heroku Deployment</strong></summary>

```bash
# Install Heroku CLI
npm install -g heroku

# Login and create app
heroku login
heroku create your-portfolio-name

# Set environment variables
heroku config:set NODE_ENV=production
heroku config:set SMTP_USER=your-email@gmail.com

# Deploy to space! 🚀
git push heroku main
heroku open
```
</details>

<details>
<summary><strong>🌊 Netlify Deployment</strong></summary>

```bash
# Install Netlify CLI
npm install -g netlify-cli

# Build and deploy
npm run build
netlify deploy --prod --dir=dist
```
</details>

## 🌐 **API Endpoints**

| Endpoint | Method | Description | Response |
|----------|--------|-------------|----------|
| `/` | GET | Main portfolio interface | HTML Page |
| `/projects` | GET | Projects showcase | HTML Page |
| `/api/contact` | POST | Mission control messages | JSON Status |
| `/api/health` | GET | System status check | JSON Health |

### **📡 Contact API Example**
